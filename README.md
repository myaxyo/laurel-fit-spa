# Laurel Fit & Spa — Premium Website & App Design

A premium women's fitness and spa website for **Laurel Fit & Spa**, Tashkent, Uzbekistan.

## Features

- 🖤 Dark luxury design (black + gold `#C9A84C`)
- 🇷🇺 Fully in Russian
- 📅 Interactive 4-step online appointment booking
- 📱 Mobile app design showcase with touch swipe
- ✨ Scroll animations with `motion/react`
- 🎨 Premium custom thin-stroke SVG icons
- 📋 Weekly class schedule timetable
- 💎 3-tier membership pricing

## Sections

1. **Hero** — cinematic full-screen with animated stats
2. **О нас** — brand story & values
3. **Услуги** — 6 service cards with hover glow
4. **Расписание** — weekly class timetable
5. **Тренеры** — trainer profiles
6. **Цены** — membership plans (Базовый / Премиум / VIP Люкс)
7. **Запись** — 4-step online booking wizard
8. **Приложение** — mobile app design mockup
9. **Отзывы** — member testimonials
10. **Footer** — contacts & links

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS v4
- motion/react (animations)
- Vite + pnpm

## Fonts

- **Playfair Display** — display headings
- **Jost** — body & UI
